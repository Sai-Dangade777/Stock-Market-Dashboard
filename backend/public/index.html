Stock Market Dashboard API

This is the backend API for the Stock Market Dashboard application.

API Endpoints

Get All Companies

GET /.netlify/functions/api/companies

Returns a list of all companies with their current stock information.

Get Company Details

GET /.netlify/functions/api/company/:symbol

Returns detailed information about a specific company.

Get Stock History

GET /.netlify/functions/api/history/:symbol

Returns historical stock data for a specific company.

Query parameters:

  • period: Time period (1d, 1w, 1m, 3m, 6m, 1y, 5y)
